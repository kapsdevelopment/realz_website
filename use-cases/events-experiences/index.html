Use case

Verified photos for events and experiences

Some moments matter more when other people can trust they were captured as real. Realz helps preserve event photos with a stronger authenticity signal than an ordinary image post.

Best fit

Concerts, conferences, meetups, competitions, trips, milestones, and other moments where proof of originality may matter later.

Why real moments need proof

Online event imagery is easy to repost, edit, generate, or remove from context. Realz gives users a way to publish a photo with a verification link so others can check the Realz proof behind it.

How Realz helps

  • Capture the event photo inside Realz.
  • Create a cryptographic proof for the captured image.
  • Share a public verification link when the photo should be trusted.
  • Keep photos private unless the user chooses to share them.

What Realz does not claim

Realz does not prove the meaning of an event or every fact around it. It provides evidence about the photo's capture and proof path.

Practical workflow

Capture the moment in Realz while it happens. If authenticity matters later, create and share the proof so others can inspect the verification state instead of relying on a caption alone.

C2PA-compatible Content Credentials

When a verified event photo is exported, Realz can add embedded, C2PA-compatible Content Credentials so provenance metadata can travel with the image file. The public Realz verify URL remains useful if the file is recompressed or metadata is removed.

Questions people ask

Can Realz replace normal social posting?

Realz is not meant to replace every camera or social app. It is useful when a moment needs a stronger proof signal.

Can I share only when I need to?

Yes. Realz is private-first. Public verification is created when the user chooses to share a proof.

Use Realz when

  • A moment may be questioned later.
  • You want more than a normal upload.
  • Proof of originality matters.
